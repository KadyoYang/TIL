# 딥 러닝

### 퍼셉트론
```
신경망의 가장 기본 단위

x1 x2  의 입력을 받고 
각각의 가중치
w1 w2 w0(bias) 계산한 다음
활성화함수(activation function)을 걸쳐서 y(출력값)을 낸다

y = activation(w0 + w1*x1 + w2*x2)

퍼셉트론은 선형 분류기로서 사용가능 

하지만 선형적으로만 분류를 할 수 있기때문에 어중간한거는 안됬음

그래서 다층 퍼셉트론이 사용

입력 레이어 -> 히든 레이어 -> 출력 레이어
히든 레이어에 여러 조건을 하는 것들 쌓는다
여러층 가능 
```


### 딥러닝
```
히든층이 많아지면 딥러닝이 됨

딥러닝 모델의 학습순서
1. 학습용 feature 데이터를 입력하여 예측값 구하기 (순전파)
2. 예측값과 실제값 사이의 오차 구하기 (Loss 계산)
3. Loss를 줄일수 있는 가중치 업데이트 하기(역전파)
4. 1~3번 반복 Loss를 최소로 하는 가중치 얻기 

```
