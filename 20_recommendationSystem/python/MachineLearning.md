# 머신러닝

### 자료 형태
```
수치형 자료(Numerical data)
 - 연속형 자료(Continuous data) : 연속적인 관측값 (원주율, 시간 )
 - 이산형 자료(Discrete data) : 셀수있는 관측값


범주형 자료(Categorical data)
 - 순위형 자료(Ordinal data) : 싫어요 보통 좋아요
 - 명목형 자료(Norminal data) : 혈액형


```


### 데이터 전 처리 
```
데이터 수집 > 데이터 분석 및 전 처리 > 머신러닝 학습 > 머신러닝 평가

1. 머신러닝의 입력 형태로 데이터 변환
2. 결측값 및 이상치를 처리하여 데이터 정제
3. 학습용 및 평가용 데이터 분리 


정규화 (Normalization)
x` = (x - x.min) / (x.max - x.min)
0.0 ~ 1.0 으로 변환됨 

표준화 (Standardication)


지도학습에서는 feature와 label이 있음
feature는 입력값
label은 예측해야할 대상이 되는 데이터 
```

### 지도학습 회귀
```
회귀분석이란 
데이터를 가장 잘 설명하는 모델을 찾아
입력값에 따른 미래 결과값을 예측 

단일 선형 회귀 : 하나의 피쳐로 계산
다중 선형 회귀 : 여러 입력값으로 계산 가능

기온과 습도를 통해서 아이스크림 판매량 예측 
```
