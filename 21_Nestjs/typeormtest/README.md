## Prerequisite

```
1. runDB.cmd 파일 안의 커맨드로 mysql 컨테이너 동작해야함
2. mysql에 접속해서 'test' 데이터베이스를 수동으로 만들어줘야함

```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
